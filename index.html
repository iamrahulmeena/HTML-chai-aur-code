we use b tag for bold the text

for strong tag
we use Strong tag for important text
strong tag also look like b tag but we use b tag only for not not any importace.

for i tag
we use i tag to display text in italics

for em tag
we use em tag to emphasize the text (same as italics).i for styling but em for making text to attention payable

for sub tag
we use sub tag to write value in base.
for example H2O

for sup tag
we use sup tag to write value in power.
for example 6X3 +3X2+10X+5
